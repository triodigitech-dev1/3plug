# Healthcare & Medical

## Module Scope
This module provides operational control, transaction integrity, and traceability for Healthcare & Medical.

## Module Dashboard
### Primary Widgets
- Throughput KPI cards
- Pending action queue
- Exception alerts
- Recent transaction timeline

### Quick Actions
- Create new record
- Continue in-progress workflow
- Escalate/approve action
- Export operational report

### Drilldowns
- By branch
- By user/cashier
- By shift/session
- By category/SKU

## Functions
- Patient registration (non-clinical ops)
- Pharmacy dispensing tracking
- Lab test billing
- Medical inventory control
- Appointment scheduling
- Prescription tracking
- Insurance claim billing workflows

- Variance vs baseline/target
- Exception concentration heatmap
- Forecast signal for next period
