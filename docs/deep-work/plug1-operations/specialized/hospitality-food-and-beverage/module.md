# Hospitality, Food & Beverage

## Module Scope
This module provides operational control, transaction integrity, and traceability for Hospitality, Food & Beverage.

## Module Dashboard
### Primary Widgets
- Throughput KPI cards
- Pending action queue
- Exception alerts
- Recent transaction timeline

### Quick Actions
- Create new record
- Continue in-progress workflow
- Escalate/approve action
- Export operational report

### Drilldowns
- By branch
- By user/cashier
- By shift/session
- By category/SKU

## Functions
- Table management
- Kitchen order ticket (KOT) system
- Recipe & ingredient tracking
- Food cost analysis
- Reservation management
- Room booking (for hotels)
- Housekeeping task tracking
- Bar tab management
- Event catering workflow management
