# Media & Creative Industries

## Module Scope
This module provides operational control, transaction integrity, and traceability for Media & Creative Industries.

## Module Dashboard
### Primary Widgets
- Throughput KPI cards
- Pending action queue
- Exception alerts
- Recent transaction timeline

### Quick Actions
- Create new record
- Continue in-progress workflow
- Escalate/approve action
- Export operational report

### Drilldowns
- By branch
- By user/cashier
- By shift/session
- By category/SKU

## Functions
- Advertising slot booking
- Production cost tracking
- Talent contract tracking
- Equipment rental scheduling
- Content sales tracking
