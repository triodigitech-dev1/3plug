# Construction & Real Estate

## Module Scope
This module provides operational control, transaction integrity, and traceability for Construction & Real Estate.

## Module Dashboard
### Primary Widgets
- Throughput KPI cards
- Pending action queue
- Exception alerts
- Recent transaction timeline

### Quick Actions
- Create new record
- Continue in-progress workflow
- Escalate/approve action
- Export operational report

### Drilldowns
- By branch
- By user/cashier
- By shift/session
- By category/SKU

## Functions
- Project cost tracking
- Site material tracking
- Equipment allocation
- Contractor billing
- Property rent collection
- Lease tracking
- Facility maintenance scheduling
