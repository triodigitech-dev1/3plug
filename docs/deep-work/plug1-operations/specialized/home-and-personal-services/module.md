# Home & Personal Services

## Module Scope
This module provides operational control, transaction integrity, and traceability for Home & Personal Services.

## Module Dashboard
### Primary Widgets
- Throughput KPI cards
- Pending action queue
- Exception alerts
- Recent transaction timeline

### Quick Actions
- Create new record
- Continue in-progress workflow
- Escalate/approve action
- Export operational report

### Drilldowns
- By branch
- By user/cashier
- By shift/session
- By category/SKU

## Functions
- Service appointment scheduling
- Mobile technician dispatch
- Service package billing
- Home visit tracking
- Subscription-based service billing

- Service appointment scheduling daily/weekly/monthly report set
- Mobile technician dispatch daily/weekly/monthly report set
- Service package billing daily/weekly/monthly report set
- Home visit tracking daily/weekly/monthly report set

- Variance vs baseline/target
- Exception concentration heatmap
- Forecast signal for next period

- Immutable audit events for writes

- Detail endpoint

- Reporting endpoints for exports
